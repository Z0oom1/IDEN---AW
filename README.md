# IDEN - Sistema de Identificação de Materiais

Bem-vindo ao **IDEN**, o Sistema de Identificação de Materiais da Alimentos Wilson. Este sistema foi desenvolvido para otimizar e agilizar o processo de identificação de produtos, garantindo precisão e eficiência nas operações de almoxarifado e gestão de estoque.

## 🚀 Otimização de Processos: Antes vs. Depois do IDEN

Antes da implementação do IDEN, a identificação manual de materiais era um processo demorado e suscetível a erros. A tarefa de identificar um caminhão inteiro de produtos, por exemplo, consumia em média **17 minutos**.

Com o IDEN, este tempo foi drasticamente reduzido para apenas **1 minuto**, representando uma **redução de 94%** no tempo gasto por identificação. Essa otimização não apenas acelera as operações, mas também minimiza a fadiga dos operadores e aumenta a produtividade geral.

### Comparativo de Eficiência

| Processo             | Tempo Anterior (minutos) | Tempo com IDEN (minutos) | Redução (%) |
| :------------------- | :----------------------- | :----------------------- | :---------- |
| Identificação Média  | 17                       | 1                        | 94%         |

## ✨ Funcionalidades Principais

*   **Catálogo de Produtos:** Gerenciamento fácil e rápido de todos os produtos, com busca e filtros inteligentes.
*   **Geração de Etiquetas:** Impressão instantânea de etiquetas padronizadas com todas as informações necessárias.
*   **Visualização 3D:** Preview interativo da etiqueta antes da impressão para garantir a conformidade.
*   **Modo Visitante:** Acesso rápido para conferentes, registrando suas atividades sem a necessidade de uma conta completa.

## 🛠️ Tecnologias Utilizadas

O IDEN é construído com tecnologias web modernas para garantir uma experiência de usuário fluida e responsiva:

*   **HTML5:** Estrutura semântica do conteúdo.
*   **CSS3:** Estilização moderna e responsiva, com temas claro e escuro.
*   **JavaScript:** Lógica de negócio e interatividade da aplicação.
*   **Node.js (Server):** Backend simples para persistência de dados (via `db.json`).

## ⚙️ Instalação e Uso

Para configurar e executar o sistema localmente, siga os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/Z0oom1/IDEN---AW.git
    cd IDEN---AW
    ```
2.  **Instale as dependências do backend (se necessário):**
    ```bash
    npm install
    ```
3.  **Inicie o servidor:**
    ```bash
    node server.js
    ```
4.  **Abra o `index.html` no seu navegador:**
    Navegue até o arquivo `index.html` no diretório raiz do projeto e abra-o com seu navegador preferido.
